# Taiko DAO CLI Tool

A lightweight CLI tool for interacting with the Taiko DAO, built with TypeScript and Node.js.

## Installation

```bash
git clone https://github.com/taikoxyz/dao-cli taiko-dao-cli
cd taiko-dao-cli
pnpm install
```
## Build
```bash
pnpm build
```

## Usage

```bash
pnpm start
```
